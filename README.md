# 한세대학교 학점 계산 도우미 (웹 프로토타입)

## 개요
한세대 학생을 위한 학점 계산/이수 학점 관리/성적 변환 기능을 한 곳에 모은 웹사이트 프로토타입입니다.  
모든 데이터는 브라우저 localStorage에만 저장되며 서버로 전송되지 않습니다.

## 주요 기능
- 학점 계산기: 학번 + 학과 입력 → 졸업 요건 충족 여부/진행률 표시
- 이수 학점 리스트: 학번 기반 시뮬레이션 수강 내역 생성 및 표로 출력
- 성적 변환 계산기: 과목별 점수 입력 기반 GPA 계산 (페이지 구현 기준)
- 강의 평가(데모): 과목 선택 시 예시 카드 출력

## 폴더 구조
- 1_assets: 파비콘/OG이미지/로고 등 정적 리소스
- 2_CSS: 공통/페이지별 스타일
- 3_JS: 공통/페이지별 스크립트
- 4_HTML: 각 메뉴 페이지

## 실행 방법
1. 저장소 다운로드
2. `WEB/index.html`을 브라우저로 실행

> 로컬에서 경로 문제가 생기면 VS Code Live Server로 실행 권장
